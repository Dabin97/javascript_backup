이벤트 : 웹페이지에서 일어나는 상황(클릭, 키보드 자판을 눌렀을때...)

마우스
    onclick : 마우스를 클릭 했을때
    ondblclick : 마우스를 더블 클릭했을때
    onmousedown : 마우스 버튼을 눌렀을때
    onmouseup : 마우스 버튼을 눌렀다가 뗀 순간
    onmousemove : 마우스를 움직였을때
    onmouseover / onmouseout : 마우스 포인터가 영역에 올라갔을때/벗어났을때
    onmouseenter / onmouseleave : 마우스 포인터가 영역에 올라갔을때/벗어났을때
키보드
    onkeydown : 키보드 키를 눌렀을때
    onkeypress : 키보드 키를 누르고 있을때
    onkeyup : 키보드 키를 눌렀다가 뗀 순간
폼 
    onchange : input 태그의 값이 바뀌었을때
    onfocus/onblur : 태그가 선택이 된 순간/선택이 해제된 순간
    onsubmit : submit 버튼을 눌러서 submit 발동되는 순간


기타
onload : 웹문서 로드가 완료 되었을때
